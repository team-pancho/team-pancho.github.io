Software and documentation

  • Latest HDG3D package. (This contains a library of Matlab functions. There is a fully developed script to test the method and a couple of three dimensional meshes. A pdf file with the full documentation of the method is also available. No installation is required.)
  • A copy of a slightly earlier version of the code in the ACM repository.
  • The ACM TOMS paper explaining how HDG3D is coded.
  • You can also visit the GitHub repository.

The research leading to the HDG3D package has been partially funded by the National Science Foundation through the grant NSF-DMS 1216356. Numerical simulation and analysis of transient waves in unbounded domains.
